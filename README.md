# gpxtools

**gpxtools** is a suite of image processing tools designed to work with the libgpx library.

 > This software project is currently under development. As Otto von Bismarck once famously said, 'Laws are like sausages, it is better not to see them being made.' The same could be said for software development, as the process can be messy and complicated. 

To include submodules, please clone this repository using the `--recurse` flag.

`git clone https://github.com/tstih/gpxtools.git --recurse`
